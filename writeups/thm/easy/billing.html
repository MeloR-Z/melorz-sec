Exploit (MagnusBilling Command Injection CVE-2023-30258)

Investigando un poco del sistema

MagnusBilling es un sistema open-source de facturación ("billing") y gestión para proveedores de telefonía IP (VoIP) que se basa en Asterisk (y tecnologías relacionadas). [MagnusBilling Wiki+2MagnusBilling+2](https://wiki.magnusbilling.org/en/source/intro.html?utm_source=chatgpt.com)

En sus propias palabras:

MagnusBilling es una herramienta completa y potente para cualquier persona que quiera comenzar como proveedor de telefonía IP en cuestión de minutos.

Descubrimos que hay un CVE publico para este sistema opensource:
