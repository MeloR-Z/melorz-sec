Como podemos ver hay un error visible que podemos aprovechar

Explotación

Vamos a hacer uso de CAST() para convertir un tipo de dato a otro y que salten errores en pantalla

1 - ' AND 1=CAST((SELECT username FROM users LIMIT 1) as int)--

este payload nos va a permitir convertir el tipo de datos que alberga la columna username a un int y va a saltar error porque el tipo de dato de username es STRING y nos mostrara el valor de username para la FILA 1 por LIMIT 1

[imagen]

2 - ' AND 1=CAST((SELECT password FROM users LIMIT 1) as int)--

ahora que ya sabemos que el username#1 es administrator podemos mostrar la password en el error con este payload, similar al otro y siguiendo la misma lógica
